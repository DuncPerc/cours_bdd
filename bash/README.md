# Cours Bash – Sommaire

Bienvenue dans ce module d’introduction au shell Bash. Ce cours est composé de plusieurs fichiers thématiques, chacun abordant un aspect fondamental de l’utilisation du terminal sous Linux.

---

## Contenu du cours

| Fichier | Thème abordé |
|--------|---------------|
| [intro.md](./intro.md) | Introduction au shell et à Bash |
| [variable_env.md](./variable_env.md) | Variables d’environnement, `export`, `echo`, `unset` |
| [vim.md](./vim.md) | Prise en main de l’éditeur `vim` |
| [xecute_bash_script.md](./xecute_bash_script.md) | Écriture et exécution de scripts Bash |

---

## Objectifs pédagogiques

- Comprendre le fonctionnement du shell Bash
- Manipuler les variables d’environnement
- Éditer des fichiers en ligne de commande avec `vim`
- Créer et exécuter des scripts Bash simples

---

## Conseils

- Utilise un terminal Linux ou un environnement comme GitHub Codespaces pour tester les commandes.
- N’hésite pas à modifier les scripts et à expérimenter.
- Chaque fichier `.md` contient des exemples pratiques et des exercices.
